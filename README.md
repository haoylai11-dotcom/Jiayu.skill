## 一些我的经验

- agency 一开始要画大饼 预算别说太细 让他们觉得你是大客户 达人量才会多
- 小频道播粉比天然高 大频道维持 5% 就算强 不能一刀切
- 已经发过我们产品内容的 KOL 永远是最高优先级
- CPM 才是核心性价比指标 不是报价绝对值
- 日本市场 KOL 密度最高竞争最小 同预算 ROI 比英语市场高
- Ambassador 长期合作（$200/月+15%）比单次付费视频（$2000+）划算 前提是人真的活跃
